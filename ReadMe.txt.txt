Sebelum menggunakan aplikasi Rula, mohon pertikan step-step berikut :

1. Pastikan kamu sudah menginstall aplikasi Xampp, download disini jika belum memiliki (https://www.apachefriends.org/download.html)

2. lalu buat database dengan nama db_rula
   ** nama tidak boleh beda **

3. Kemudian buka database "db_rula" yang dibuat, lakukan import menggunakan file didalam folder _IMPORTANT!

4. Done! Selamat menggunakan aplikasi RULA
